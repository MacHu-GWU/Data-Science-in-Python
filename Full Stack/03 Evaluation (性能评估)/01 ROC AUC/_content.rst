模型评估 (Model Evaluation)
==============================================================================


二分类问题中的AUC曲线
------------------------------------------------------------------------------
面对二分类问题, 我们得到一个模型, 对于每一个样本, 能够输出一个值 y = f(x), 当y大于某个阈值 t 时, 我们判定为P, 反之判定为F.

Confusion Matrix::

    实际\预测   P   N
        +       TP  TN
        -       FP  FN

- ``TPR = TP / (TP + FN)``
- ``FPR = FP / (FP + TN)``

对于从正无穷到负无穷的t, 我们可以以FPR为横轴, TPR为纵轴画出一条曲线, 这条曲线就叫做 . 而距离(0, 1)最近的哪个点所对应的t, 通常就是最理想的阈值.
