Install
------------------------------------------------------------------------------

``data_science_in_python`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install data_science_in_python

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade data_science_in_python
